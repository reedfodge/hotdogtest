This code is used to determine if a user selected local image is the same image as the hot dog image attached.

Instructions:
> Run the code
> Use File -> Upload Image to select image you want compared
> Click on "Test" button to determine if your image is a hot dog
> Selected the wrong image? Go to File -> Reset to clear image

Instructions also avaliable via Help -> Instructions

Note: Inspiration came from the HBO show "Silicon Valley"
